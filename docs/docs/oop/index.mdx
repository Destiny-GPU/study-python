# 🏗️ 面向对象

飞行器是一个"对象"——它有属性（姿态、速度）和方法（加速、转向）。面向对象编程让你用这种方式组织代码，让复杂系统变得清晰可管理。

:::tip[🎯 本章目标]
学完本章，你将能：
- 定义类和创建对象，理解 `__init__` 和 `self`
- 用继承建立类型层次（如：滤波器 → 低通滤波器 → 巴特沃斯滤波器）
- 用 `@dataclass` 简化数据容器定义
- 用魔术方法让自定义类支持 `len()`、`+`、`in` 等操作
:::
