# 🧩 模块与包

当代码量超过几百行，单文件结构就力不从心了。Python 的模块与包机制让你按文件拆分代码、按目录组织依赖，用 `import` 一行语句就能复用任意模块。

:::tip[🎯 本章目标]
学完本章，你将能：
- 用 `import` 和 `from...import` 导入模块，理解 `__name__` 机制
- 用包（package）组织多模块项目，掌握 `__init__.py` 与命名空间包
- 用 `pip` 和 `uv` 管理第三方依赖
- 用虚拟环境隔离项目依赖，避免版本冲突
:::
