# 🔀 控制流

程序的执行路径并非一成不变——有时需要根据条件选择不同分支，有时需要重复执行某段代码。控制流就是让程序"做决定"和"重复做事"的机制。

:::tip[🎯 本章目标]
学完本章，你将能：
- 用 `if-else` 根据条件执行不同逻辑（如：根据传感器读数判断飞行状态）
- 用 `for` 循环遍历数据序列（如：遍历一批飞行实验数据）
- 用 `while` 循环处理持续条件（如：等待系统收敛）
- 用 `match-case` 进行模式匹配（如：解析不同的飞行指令）
:::
