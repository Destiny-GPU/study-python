# 🛡️ 异常处理

程序运行中难免遇到错误——文件不存在、网络超时、除零、类型不匹配。异常处理让你的程序在遇到问题时优雅恢复，而不是直接崩溃。

:::tip[🎯 本章目标]
学完本章，你将能：
- 用 `try-except` 捕获和处理异常
- 理解 Python 异常层次结构，选择正确的捕获粒度
- 自定义异常类，表达业务错误语义
- 用上下文管理器（`with`）自动管理资源
:::
