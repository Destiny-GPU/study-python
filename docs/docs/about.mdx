## 为什么创建 Study Python？

在学习 Python 的过程中，我发现大多数教程都用"Hello World"、计算器、猜数字这样的玩具例子来教学。这些例子虽然简单，但离真实应用太远——**学完之后，你还是不知道怎么用 Python 解决实际问题**。

作为自动控制方向的博士生，我每天都在用 Python 做信号处理、数据分析、算法验证。我深知：**真正掌握一门语言，需要看到它在真实场景中的样子**。

所以我想做一件事：

> 用飞行器智能控制领域的真实案例，重新讲解 Python 的每一个知识点。  
> 让你学到的不只是语法，而是**能直接用在科研和工程中的技能**。

这个项目还在不断完善中。如果你觉得有帮助，欢迎 Star 或分享给更多人。
