## 如何使用

在任何教程 MDX 文件中添加挑战：

```jsx
import ControlChallenge from '@site/src/components/ControlChallenge';

<ControlChallenge
  title="挑战标题"
  scenario="场景描述..."
  objective="你要完成的任务..."
  starterCode={`# 起始代码`}
  solution={`# 参考答案`}
  tests={[
    { call: 'function_name(args)', expected: [min, max], label: '测试描述' },
  ]}
/>
```

:::tip
- `tests` 中的 `expected` 是 `[最小值, 最大值]` 范围，允许浮点误差
- `solution` 可选，不提供则不显示"查看答案"按钮
- `starterCode` 中用 `# TODO` 标记需要学生填写的位置
:::
