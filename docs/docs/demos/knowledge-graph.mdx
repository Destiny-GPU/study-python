# 知识脉络图

一张图看清整个 Python 学习路径。**拖拽平移、滚轮缩放、点击跳转**。

<KnowledgeGraph />

## 如何阅读

- **节点** = 一个知识模块，颜色代表所属阶段
- **箭头** = 前置依赖关系（学 A 之前要先学 B）
- **左侧色条** = 模块的分类颜色

## 五个阶段

| 阶段 | 颜色 | 内容 |
|---|---|---|
| 入门准备 | 🟢 绿色 | Python 简介、环境搭建 |
| 核心语法 | 🔵 蓝色 | 基础语法、流程控制、数据结构、函数 |
| 进阶概念 | 🟡 橙色 | 面向对象、模块、异常处理、文件操作 |
| 高级主题 | 🟣 紫色 | 标准库、NumPy/SciPy/pandas/PyTorch、装饰器、异步编程 |
| 工程实践 | 🩷 粉色 | 测试调试、并行计算、性能优化、项目实战 |

:::tip
点击顶部的分类标签可以过滤显示特定阶段的知识点。再次点击取消过滤。
:::
