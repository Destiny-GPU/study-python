# ⚙️ 函数

函数是代码复用的基石。把重复的逻辑封装成函数，调用一次就能用无数次——这是从"写脚本"到"写程序"的关键一步。

:::tip[🎯 本章目标]
学完本章，你将能：
- 定义和调用函数，理解参数传递机制
- 用装饰器在不修改原函数的情况下扩展功能
- 用生成器处理大规模数据流（如实时传感器数据）
- 理解作用域链（LEGB），避免常见的变量陷阱
:::
