# 🧰 标准库

Python 的"自带电池"——标准库覆盖了操作系统交互、路径处理、时间日期、高效容器、迭代工具等常见需求，无需安装第三方包即可使用。

:::tip[🎯 本章目标]
学完本章，你将能：
- 用 `os` 和 `pathlib` 进行文件系统操作
- 用 `datetime` 处理时间日期与时间区
- 用 `collections` 的专用容器（deque、Counter、defaultdict）简化代码
- 用 `itertools` 写出高效的迭代逻辑
- 用 `functools` 缓存计算、偏函数、类型分派
:::
