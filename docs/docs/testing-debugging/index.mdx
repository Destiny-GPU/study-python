# 🔍 测试与调试

写完代码怎么知道它对不对？出了 bug 怎么快速定位？测试和调试是保障代码质量的两大支柱。

:::tip[🎯 本章目标]
学完本章，你将能：
- 用 `pytest` 编写和运行测试用例
- 掌握参数化测试、fixture、Mock 等进阶技巧
- 用 `logging` 替代 print 实现分级日志
- 用 pdb 和 debugpy 进行远程调试
:::
