# 🛠️ 入门准备

万事开头难？其实搭建 Python 环境只需要 5 分钟。本章帮你从零搭建开发环境，写出第一个程序，并理解代码的多种运行方式。

:::tip[🎯 本章目标]
学完本章，你将能：
- 在你的操作系统上安装 Python 3.12+
- 用 uv 管理包依赖（比 pip 快 10-100 倍）
- 运行第一个 Python 程序
- 配置 VS Code 开发环境
:::

- [安装 Python 环境](./getting-started/installation) — 在 Linux、Windows、macOS 上安装 Python 3.12+，掌握 pyenv 与 uv 包管理
- [第一个 Python 程序](./getting-started/first-program) — 编写并运行第一个程序，认识 print、注释与 REPL 交互环境
- [运行 Python 代码的方式](./getting-started/running-code) — 脚本、REPL、`python -c`、`__main__` 入口、uv run 等多种运行方式
- [IDE 配置指南](./getting-started/ide-setup) — 配置 VS Code 与 PyCharm，集成 uv 环境管理工具
