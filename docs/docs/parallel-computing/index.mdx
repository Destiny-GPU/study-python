# ⚡ 并行计算

Python 的 GIL 让多线程无法利用多核 CPU——但多进程和正确的并行策略可以突破这个限制。本章教你根据任务类型选择最佳并行方案。

:::tip[🎯 本章目标]
学完本章，你将能：
- 理解 GIL 的本质及其对并行的影响
- 用 multiprocessing 利用多核 CPU 加速计算
- 用 threading 处理 I/O 密集任务（如网络请求）
- 用 concurrent.futures 写出简洁的并行代码
:::
