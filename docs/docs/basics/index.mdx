# 📝 基础语法

变量、字符串、数字、布尔值——这些是编程的"字母表"。掌握它们，你就能写出有意义的 Python 程序。

:::tip[🎯 本章目标]
学完本章，你将能：
- 声明变量并理解 Python 的动态类型机制
- 使用 f-string 格式化输出（如打印飞行器姿态角）
- 掌握数字、字符串、布尔值等基本类型
- 用运算符进行计算和比较
:::
