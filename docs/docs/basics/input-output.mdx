输入与输出（I/O）是程序与外界交互的桥梁。Python 提供了简洁强大的 `print()` 与 `input()` 函数，以及灵活的标准流（`sys.stdin` / `sys.stdout` / `sys.stderr`）。本节我们将一起探索它们的高级用法、多种格式化输出方式，并介绍用于终端美化的 `rich` 库。

## print() 函数

`print()` 是 Python 中最常用的输出函数，它有许多实用的参数：

```py title="Python"
print?
# 签名：print(*objects, sep=' ', end='\n', file=sys.stdout, flush=False)
```

| 参数    | 含义                              | 默认值       |
| ------- | --------------------------------- | ------------ |
| `*objects` | 要打印的内容（可多个）         | -            |
| `sep`   | 多个对象之间的分隔符             | `' '`（空格）|
| `end`   | 末尾追加的字符                    | `'\n'（换行）`|
| `file`  | 输出到的文件对象                  | `sys.stdout` |
| `flush` | 是否立即刷新输出缓冲区            | `False`      |

### sep 参数：分隔符

```py title="Python"
# 默认用空格分隔
print("a", "b", "c")  # a b c

# 自定义分隔符
print("2024", "01", "15", sep="-")   # 2024-01-15
print("Alice", "Bob", "Charlie", sep=", ")  # Alice, Bob, Charlie
print("a", "b", "c", sep="")  # abc (无分隔)
print("序号", "姓名", "成绩", sep="\t")  # 序号  姓名  成绩 (制表符分隔)
print("a", "b", "c", sep="\n")  # 每个元素换行
```

### end 参数：结尾字符

```py title="Python"
# 默认换行
print("第一行")
print("第二行")

# 不换行
print("Hello", end=" ")
print("World")  # Hello World

# 自定义结尾
for i in range(5):
    print(i, end=" ")  # 0 1 2 3 4 (同行，空格分隔)
print()  # 最后换行

# 进度条效果（用 \r 回到行首）
import time

for i in range(101):
    print(f"\r进度: {i}%", end="")
    time.sleep(0.01)
print("\n完成！")
```

:::tip[\r 与进度条]
`end="\r"`（回车符）让光标回到行首而不换行，是实现终端进度条、动态刷新的核心技巧。结合 `end=""` 不换行，可创建动态更新的输出。
:::

### file 参数：输出到文件

```py title="Python"

with open("output.txt", "w", encoding="utf-8") as f:
    print("写入文件的第一行", file=f)
    print("写入文件的第二行", file=f)

# 同时输出到屏幕和文件（典型日志场景）
import sys

class Tee:
    """同时写入多个流"""
    def __init__(self, *streams):
        self.streams = streams

    def write(self, data):
        for s in self.streams:
            s.write(data)

    def flush(self):
        for s in self.streams:
            s.flush()
```

### flush 参数：立即刷新

```py title="Python"
import time

# 默认情况下，print 会缓冲输出
# flush=True 强制立即输出（用于实时显示）
for i in range(3):
    print(f"处理中... {i+1}/3", flush=True)
    time.sleep(0.5)
```

:::info[何时需要 flush=True？]
在以下场景中，输出可能不会立即显示，需要 `flush=True`：
1. **管道输出**：`python script.py | grep xxx` 时，stdout 是块缓冲的
2. **进度条/实时刷新**：确保信息立即显示
3. **日志输出**：避免程序崩溃时丢失最后的日志
4. **GUI/IDE 环境**：某些 IDE 的输出窗口有缓冲
:::

## input() 函数

`input()` 用于从标准输入读取一行文本，返回**字符串**：

```py title="Python"
# 基本用法
name = input("请输入你的名字：")
print(f"你好，{name}！")

# 注意：input 总是返回字符串，需要手动转换类型
age_str = input("请输入你的年龄：")
age = int(age_str)  # 转为整数
print(f"明年你 {age + 1} 岁")

# 更安全的转换
try:
    height = float(input("请输入你的身高（米）："))
    print(f"你的身高是 {height} 米")
except ValueError:
    print("输入的不是有效数字")
```

:::warning[input() 永远返回字符串]
`input()` 不管你输入什么，返回的都是字符串。输入 `123` 得到的是 `"123"` 而不是数字 `123`，需要用 `int()` 或 `float()` 转换。忘记转换是初学者的常见错误。
:::

### 封装安全的输入函数

```py title="Python"
def input_int(prompt: str, default: int | None = None) -> int:
    """安全读取整数输入"""
    while True:
        s = input(prompt)
        if not s and default is not None:
            return default
        try:
            return int(s)
        except ValueError:
            print("请输入有效的整数")


def input_float(prompt: str, default: float | None = None) -> float:
    """安全读取浮点数输入"""
    while True:
        s = input(prompt)
        if not s and default is not None:
            return default
        try:
            return float(s)
        except ValueError:
            print("请输入有效的数字")


def input_choice(prompt: str, options: list[str]) -> str:
    """让用户从选项中选择"""
    while True:
        s = input(prompt).strip()
        if s in options:
            return s
        print(f"请从以下选项中选择: {options}")


# 使用示例
# age = input_int("请输入年龄（默认 18）：", default=18)
# print(f"你的年龄: {age}")
```

## 格式化输出

Python 有三种主要的字符串格式化方式。

### 1. f-string（推荐）

```py title="Python"
name = "Alice"
age = 25
height = 1.68
pi = 3.14159

# 基本用法
print(f"我叫 {name}，今年 {age} 岁")

# 表达式
print(f"明年我 {age + 1} 岁")

# 格式化数字
print(f"身高: {height:.2f} 米")      # 1.68
print(f"圆周率: {pi:.4f}")          # 3.1416
print(f"成绩: {85.5:>10.2f}")       # 右对齐宽度 10
print(f"成绩: {85.5:<10.2f}")       # 左对齐
print(f"成绩: {85.5:^10.2f}")       # 居中

# 千位分隔符
big_num = 1234567890
print(f"{big_num:,}")    # 1,234,567,890
print(f"{big_num:_}")    # 1_234_567_890

# 百分比
ratio = 0.856
print(f"通过率: {ratio:.2%}")  # 85.60%

# 科学计数法
print(f"{123456789:.2e}")  # 1.23e+08

# 不同进制
num = 255
print(f"十进制: {num}")        # 255
print(f"二进制: {num:b}")       # 11111111
print(f"八进制: {num:o}")       # 377
print(f"十六进制: {num:x}")     # ff
print(f"十六进制(大写): {num:X}")  # FF

# 日期格式化
from datetime import datetime
now = datetime.now()
print(f"现在时间: {now:%Y-%m-%d %H:%M:%S}")  # 2024-01-15 14:30:25

# 调试输出（Python 3.8+：{x=} 等价于 x=value）
x = 42
y = "hello"
print(f"{x=}, {y=}")  # x=42, y='hello'

# Python 3.12+ 嵌套引号
user = {"name": "Alice", "age": 25}
print(f"用户: {user["name"]} ({user["age"]} 岁)")
```

## 对齐与表格输出

利用格式化语法可以制作简单的对齐表格：

```py title="Python"
# 文本对齐
def print_table(headers: list[str], rows: list[list[str]]) -> None:
    """打印简单的对齐表格"""
    # 计算每列最大宽度
    widths = [len(h) for h in headers]
    for row in rows:
        for i, cell in enumerate(row):
            widths[i] = max(widths[i], len(str(cell)))

    # 打印表头
    header_line = " | ".join(f"{h:<{widths[i]}}" for i, h in enumerate(headers))
    print(header_line)
    print("-" * len(header_line))

    # 打印数据行
    for row in rows:
        print(" | ".join(f"{str(c):<{widths[i]}}" for i, c in enumerate(row)))


# 示例
print_table(
    ["姓名", "年龄", "城市", "职业"],
    [
        ["Alice", "25", "北京", "工程师"],
        ["Bob", "30", "上海", "设计师"],
        ["Charlie", "28", "广州", "产品经理"],
    ],
)
```

## sys.stdin / stdout / stderr

### 进度条实现

```py title="Python"
import sys
import time

def progress_bar(total: int, width: int = 30) -> None:
    """演示进度条"""
    for i in range(total + 1):
        percent = i / total
        filled = int(width * percent)
        bar = "█" * filled + "-" * (width - filled)
        sys.stdout.write(f"\r[{bar}] {percent:.1%} ({i}/{total})")
        sys.stdout.flush()
        time.sleep(0.02)
    print("\n完成！")


progress_bar(50)
```

## 实战：交互式成绩录入系统

下面用一个综合示例演示 input/output 与格式化输出的实际应用：

```py title="Python"
"""交互式成绩录入系统

演示 input/print 的综合应用，含数据验证与表格输出
"""

from typing import NamedTuple


class Student(NamedTuple):
    """学生记录"""
    name: str
    chinese: float
    math: float
    english: float

    @property
    def total(self) -> float:
        return self.chinese + self.math + self.english

    @property
    def average(self) -> float:
        return self.total / 3


def input_float(prompt: str, min_val: float = 0, max_val: float = 100) -> float:
    """安全读取浮点数"""
    while True:
        try:
            value = float(input(prompt))
            if min_val <= value <= max_val:
                return value
            print(f"  错误：数值需在 {min_val}-{max_val} 之间")
        except ValueError:
            print("  错误：请输入有效数字")


def input_student() -> Student:
    """录入一个学生信息"""
    name = input("请输入姓名：").strip()
    chinese = input_float("  语文成绩：")
    math = input_float("  数学成绩：")
    english = input_float("  英语成绩：")
    return Student(name, chinese, math, english)


def print_report(students: list[Student]) -> None:
    """打印成绩报告表"""
    # 表头
    headers = ["姓名", "语文", "数学", "英语", "总分", "平均分", "等级"]
    widths = [8, 6, 6, 6, 8, 8, 6]

    # 分隔线
    sep = "+" + "+".join("-" * (w + 2) for w in widths) + "+"

    print("\n" + "=" * 50)
    print("成绩报告表".center(50))
    print("=" * 50)
    print(sep)
    print("| " + " | ".join(f"{h:<{widths[i]}}" for i, h in enumerate(headers)) + " |")
    print(sep)

    # 数据行
    for s in students:
        grade = (
            "A" if s.average >= 90
            else "B" if s.average >= 80
            else "C" if s.average >= 70
            else "D" if s.average >= 60
            else "F"
        )
        row = [
            s.name,
            f"{s.chinese:.1f}",
            f"{s.math:.1f}",
            f"{s.english:.1f}",
            f"{s.total:.1f}",
            f"{s.average:.2f}",
            grade,
        ]
        print("| " + " | ".join(f"{c:<{widths[i]}}" for i, c in enumerate(row)) + " |")

    print(sep)

    # 统计信息
    if students:
        avg_total = sum(s.total for s in students) / len(students)
        best = max(students, key=lambda s: s.total)
        print(f"\n班级平均总分: {avg_total:.2f}")
        print(f"最高分: {best.name} ({best.total:.1f} 分)")


# 主程序
def main() -> None:
    print("=== 学生成绩录入系统 ===")
    print("输入学生信息，姓名为空时结束录入\n")

    students: list[Student] = []
    while True:
        name = input(f"第 {len(students) + 1} 位学生姓名（空结束）：").strip()
        if not name:
            break
        # 临时修改 input_student 不再问姓名
        chinese = input_float("  语文成绩：")
        math = input_float("  数学成绩：")
        english = input_float("  英语成绩：")
        students.append(Student(name, chinese, math, english))
        print(f"  ✓ 已录入 {name}\n")

    if students:
        print_report(students)
    else:
        print("未录入任何学生信息")


# 交互式运行（取消注释来使用）
# if __name__ == "__main__":
#     main()

# 演示数据输出
demo_students = [
    Student("Alice", 95, 88, 92),
    Student("Bob", 78, 85, 80),
    Student("Charlie", 65, 72, 58),
]
print_report(demo_students)
```

## ✅ 本节总结

- **`print()` 函数**有 4 个关键参数：`sep`（分隔符）、`end`（结尾符）、`file`（输出目标）、`flush`（立即刷新）
- **`input()` 函数**始终返回字符串，需要手动转换类型；推荐封装安全输入函数处理异常
- **格式化**：新代码一律使用 f-string，`format()` 和 `%` 格式化仅在旧代码中出现
- f-string 支持丰富的格式规范：对齐、宽度、精度、千位分隔符、百分比、科学计数法、进制、日期等
- **`{x=}` 调试语法**（Python 3.8+）能同时输出变量名和值，调试利器
- **Python 3.12+** f-string 支持嵌套相同引号、多行表达式与注释
- `sys.stdin` / `sys.stdout` / `sys.stderr` 是标准流，可重定向用于日志、文件输出等场景
- **`rich` 库**提供终端彩色、表格、进度条等美化输出，是现代 CLI 工具的首选
- 文件 I/O 使用 `with` 语句自动管理资源，详细内容见"文件操作"章节
