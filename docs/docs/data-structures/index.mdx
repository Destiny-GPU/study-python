# 📦 数据结构

列表存一批传感器读数，字典映射配置参数，集合去重实验编号——数据结构是组织数据的骨架。

:::tip[🎯 本章目标]
学完本章，你将能：
- 用列表存储和操作有序数据（如时间序列）
- 用字典管理键值配置（如模型超参数）
- 用集合进行去重和集合运算
- 用推导式一行代码生成复杂数据结构
:::

:::info[📋 前置知识]
学习本章前，建议先完成：
- [基础语法](./basics) — 变量、数据类型、运算符
:::

- [列表（List）](./data-structures/lists) — 可变有序序列，支持灵活的增删改查操作
- [元组（Tuple）](./data-structures/tuples) — 不可变序列，适用于固定数据聚合与多返回值
- [字典（Dict）](./data-structures/dictionaries) — 可变键值映射，支持合并运算符与 defaultdict/Counter
- [集合（Set）](./data-structures/sets) — 无序不重复容器，支持并交差等集合运算
- [推导式（Comprehension）](./data-structures/comprehensions) — 一行简洁语法构造列表、字典、集合与生成器
