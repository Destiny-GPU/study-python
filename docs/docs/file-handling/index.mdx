# 📂 文件与 IO

数据需要持久化存储，程序需要读取配置、保存结果、处理用户数据。文件操作是连接程序与外部世界的桥梁。

:::tip[🎯 本章目标]
学完本章，你将能：
- 用 `open()` 和 `pathlib` 读写文本与二进制文件
- 用 `json` 模块序列化与反序列化数据
- 用 `csv` 模块处理表格数据
- 掌握现代路径处理（pathlib）替代传统 os.path
:::
